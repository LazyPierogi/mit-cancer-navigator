# Roadmap

Last updated: 2026-03-15

## Near-term

- Keep deterministic analysis, imports, and benchmark flows stable and easy to reproduce.
- Expand regression coverage for corpus validation and normalization.
- Improve contributor-facing setup, release, and troubleshooting guidance.

## Mid-term

- Harden semantic corpus indexing and retrieval storage beyond the current lab posture.
- Improve provenance metadata for imported batches and evaluation outputs.
- Reduce overlap between local scripts and runtime fallback paths.

## Longer-term

- Revisit external vector backend support after public-repo setup is fully stabilized.
- Add richer contributor workflows, release automation, and CI checks.
- Formalize governance and collaboration files for wider external participation.
